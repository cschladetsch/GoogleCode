Introduction

This is a simple command-line utility that reads HLSL .fx files, parses the result into an AST, then walks the tree producing StringTemplates. These are then used to write the HLSL back out to a text file.

Basically, it is a really long-winded way of doing a file copy.

Usage

fxparser -fx -i some_hlsl.fx

Full details on options are in the Options class in Program.cs.
